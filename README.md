## вљЎ Quick Start

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Place your model files

```bash
cp /path/to/model.pkl       ml_models/model.pkl
cp /path/to/paysim_model.pkl ml_models/paysim_model.pkl
```

### 3. Start Neo4j

```bash
docker run -d \
  --name neo4j-fraud \
  -p 7474:7474 -p 7687:7687 \
  -e NEO4J_AUTH=neo4j/frauddetection123 \
  neo4j:latest
```

Or open Neo4j Desktop and create a database with the same password.

### 4. Configure environment

```bash
cp .env.example .env
# Edit .env with your Neo4j password if different
```

### 5. Generate sample data (optional)

```bash
python scripts/generate_fraud_dataset.py --rows 10000 --fraud-rate 0.05
```

## рџ”¬ Fraud Patterns Detected

| Pattern | How detected |
|---------|-------------|
| **Fraud Rings** | Cyclic paths (3вЂ“5 hops) where all edges are fraud |
| **Mule Chains** | Linear paths (4вЂ“6 hops) of fraud transactions |
| **Coordinated Hubs** | Accounts receiving fraud from 3+ unique senders |
| **Shared Devices** | Devices used by 2+ different accounts |
| **Risk Score** | Weighted graph metric per account |
